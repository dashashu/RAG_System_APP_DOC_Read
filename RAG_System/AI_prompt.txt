1. Core RAG service
Build a Python RAG pipeline that: recursively loads PDFs from DATA_DIR, extracts text with pypdf, chunks with configurable size/overlap, embeds with OpenAI text-embedding-3-small, stores in Chroma (cosine) with precomputed embeddings (not Chroma’s OpenAI embedder). For Q&A: optional gpt-4o query expansion → multi-query embed → retrieve → merge by chunk id (best similarity) → answer with gpt-4o using context-only instructions; return answer + sources. Use pydantic-settings + .env.example.

2. FastAPI application
FastAPI with lifespan ingest (skip if already indexed; RAG_FORCE_REINDEX to force). Endpoints: GET /health, POST /query (question, use_expansion), POST /reindex (force), POST /expand-only.

3. OpenAI client and TLS
Dedicated OpenAI client using httpx with SSLContext merging certifi + common system CA paths; document SSL_CERT_FILE.

4. Docker + dependencies
requirements.txt, Dockerfile (Python 3.12-slim, non-root, DATA_DIR / CHROMA_DIR, uvicorn 8080), .dockerignore, .gitignore (.env, chroma, venv, plots).

5. Kubernetes
Deployment, Service, PVC for Chroma, secret.example.yaml (placeholders only).

6. Cross-encoder reranking
Wide Chroma pool per query → cap candidates → CrossEncoder (cross-encoder/ms-marco-MiniLM-L-6-v2) → top llm_context_top_k to the LLM; RERANK_ENABLED and related env vars; fallback if rerank fails.

7. DPR retrieval modes
Package under app/retrieval/ with RETRIEVAL_MODE: openai_chroma, dpr_only, parallel_dpr_openai (dual Chroma collections, merge by max score in parallel mode); lazy-load DPR; forced reindex when switching openai vs dpr-only; transformers 4.x compatible with chromadb’s tokenizers pin.

8. Confluence ingestion
When CONFLUENCE_ENABLED and credentials/spaces are set, call Confluence REST (/rest/api/content, pages, body.storage), strip to text, same chunking/indexes as PDFs; metadata source_type, page_id, space_key, url; confluence_pages on reindex; update prompts/sources for URLs. Support Cloud (…/wiki) vs Server base URL notes.

9. Optional viz
requirements-viz.txt + scripts for PCA/t-SNE embedding and query-context scatter plots; document in readme.

10. Documentation
ReadMe (quick start, env, API, Confluence, retrieval modes, reranking, TLS, Docker/K8s, layout).
